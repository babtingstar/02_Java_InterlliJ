
중복 데이터를 허용하지 않고, 저장된 요소의 순서를 유지하지 않는 자료

데이터를 효율적으로 저장하고 검색하기 위해 사용됨

주요 특징

1. 동일한 값이 이미 존재하면 추가되지 않음

2. 저장된 순서를 유지하지 않고, 내부적으로 Hasing 기법 사용

3. null 값 허용 = 아무것도 없는 값을 저장하는 것을 허용, null 역시 하나의 값으로 취급(최초 1회)




주요 메서드

add(E e) : 리스트 끝에 요소를 추가

remove(Object o) : 지정된 요소를 삭제(첫 번째 발견된 값만)

size() : 리스트의 크기를 반환

imEmpty() : 리스트가 비어있는지 확인

contains(Object o) : 리스트에 특정 요소가 포함되어 있는지 확인

indexOf(Object o) : 지정된 요소의 첫 번째 인덱스를 반환

clear() : 리스트의 모든 요소를 삭제

addAll(Collection c) : 다른 컬렉션의 모든 요소를 복사해서 추가

iterator() : Iterator 를 반환해 순차적으로 회전(for 문 대신 이용) 가능
